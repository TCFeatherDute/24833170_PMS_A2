🛡️ Privacy & Security Statement

Data Privacy Principle

This system strictly follows the principle of user privacy protection. All product information and data entered by users are only used for the operation of this inventory management system. We do not collect any personal identity information, and we do not obtain any private data such as name, phone number, address, or account password.

All data is processed in a secure and closed environment to ensure that user information is not leaked, abused, or accessed by unauthorized third parties.

Data Storage Method

All data in this system is stored locally in the client-side service. Data will not be uploaded to any remote server, cloud platform, or external API. Data only exists inside the application and will not be transmitted over the network.

When the page is refreshed or the project is restarted, data will be restored to the default initial state, ensuring that no residual information is stored locally for a long time.

Data Security Protection

• Unique item ID verification to prevent data duplication and confusion

• Complete form validation to avoid invalid or malicious data

• No database connection, no external network interaction, reducing security risks

• Front-end-only data processing mode to improve overall system security

• Clear data operation logic, making data behaviour controllable and traceable

User Rights & Security Commitment

Users have full control over all data in the system. Users can add, modify, delete, query, and filter data at any time. The system does not store any hidden data or automatically collect information.

We promise that the system will not use user data for commercial purposes, will not share data with third parties, and will not perform any behaviour that endangers information security.

Application Security Design

This system adopts standard Angular security specifications, including safe form operations, independent component design, data encapsulation through service, and reasonable data interaction logic.

The interface design is simple and clear, avoiding misleading operations. The system has clear error prompts and operation feedback to ensure that users can use it safely and stably.

Summary of Privacy Policy

This inventory management system is a secure, front-end-only application that focuses on privacy protection and data security. It does not collect personal information, does not upload data, does not have network risks, and provides users with a safe, reliable, and lightweight management experience.

All designs comply with basic network security and personal privacy protection specifications, fully meeting the security requirements of the course assignment.
