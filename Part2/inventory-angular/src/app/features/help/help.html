How to add an item?

Fill all required fields and click Add Item.

Why can’t I add an item?

Item ID must be unique. All required fields must be filled.

How to edit an item?

Enter the item name, modify the details, then click Edit.

How to delete an item?

Enter the item name and click Delete. Confirm when prompted.

How to search items?

Enter a keyword in the search box and click Search.
