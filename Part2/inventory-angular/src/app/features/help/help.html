Help & User Guide

About This System

This inventory management system is designed to help users manage product information easily. You can add, edit, delete, search and view products. All data is stored locally in the browser and will not be uploaded to any server.

How to Add an Item

1. Go to the Inventory page

2. Fill in all required fields: Item ID, Item Name, Supplier Name

3. Quantity and Price are set to 0 by default, you can change them

4. Category, Stock Status and Popular can be selected directly

5. Comment is optional, you can leave it empty

6. Click Add Item to save

How to Edit an Item

1. Enter the Item Name of the product you want to modify

2. Change the information you want to update

3. Click Edit Item

4. The system will update the item matching the name

How to Delete an Item

1. Enter the Item Name of the product you want to delete

2. Click Delete Item

3. Confirm the deletion in the pop-up window

4. The item will be removed permanently

How to Search Items

1. Go to the Search page

2. Enter any keyword in the search box

3. Click Search to find matching items

4. Click Popular Items to view popular products

Common Error Explanations

"All fields except Comment are required!"
→ Item ID, Item Name, Supplier Name cannot be empty.

"Item ID already exists!"
→ This ID has been used, please use a new unique ID.

"Item not found"
→ The item name you entered does not exist in the system.

Notes

• All data is temporary and stored only in the service

• Refreshing the page will restore the default 20 items

• Item ID must be unique to avoid conflict

• All forms support real‑time input and immediate operation

• The interface is responsive and user‑friendly
