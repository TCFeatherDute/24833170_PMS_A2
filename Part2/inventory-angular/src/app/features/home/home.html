Inventory Management System

PROG2005 Assignment 2 - Angular

System Introduction

This is a complete inventory management system developed with Angular. It supports item adding, editing, deleting, searching and filtering popular items. All data is stored locally on the client side.

Main Functions

  • Add new inventory items
  • Edit existing items
  • Delete items by name
  • Search items by name
  • Show all items
  • Show popular items
  • Form validation & user feedback

How to Use

1. Go to Inventory page to manage items

2. Fill all fields except Comment to add items

3. Use Search page to quickly find products

4. View popular products with one click

5. All data is stored locally in the service
